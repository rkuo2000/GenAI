# 幼兒園日語假名學習網頁 - 規格書

## 1. 概念與願景

一個專為幼兒園小朋友設計的互動式日語假名學習網頁。採用可愛、友善的視覺風格，大尺寸按鈕配合明亮的色彩，讓孩子在玩樂中輕鬆學習日語五十音。每個假名配有精心挑選的簡單單字，搭配生動圖片和語音朗讀，創造沉浸式學習體驗。

## 2. 設計語言

### 美學方向
採用「幼兒教育玩具」風格 - 圓潤的形狀、柔和的色彩、友善的字體，營造安全温暖的學習環境。

### 色彩系統
- 主色：`#FF6B9D`（櫻花粉）
- 次要色：`#4ECDC4`（薄荷綠）
- 強調色：`#FFE66D`（陽光黃）
- 背景色：`#FFF5F8`（淡粉白）
- 文字色：`#2D3436`（深灰）
- 成功色：`#00B894`（翡翠綠）

### 字體
- 標題：Noto Sans JP（粗體）
- 假名/日文：Noto Sans JP
- 英文說明：Quicksand

### 動效哲學
- 按鈕點擊：scale(0.95) → scale(1.05) → scale(1)，配合彈跳效果
- 卡片進場：fadeIn + slideUp，stagger 100ms
- 語音播放時：圖標脈動動畫
- 滑鼠懸停：輕微放大 + 陰影加深

### 圖標
使用 emoji 作為圖示（適合幼兒理解）

## 3. 版面與結構

### 頁面結構
1. **頂部標題區** - 大標題「日語假名學習」配櫻花裝飾
2. **假名選擇器** - 大尺寸按鈕，可切換「全部」「清音」「濁音」
3. **主學習區** - 網格卡片佈局展示所有假名
4. **詳情彈窗** - 點擊假名後展開的學習面板

### 響應式策略
- 桌面 (>1024px)：5-6 列網格
- 平板 (768-1024px)：4 列網格
- 手機 (<768px)：2-3 列網格
- 卡片尺寸根據螢幕調整，確保觸控友好

## 4. 功能與互動

### 核心功能
1. **假名卡片展示**
   - 顯示平假名和片假名（大字體）
   - 標準羅馬拼音
   - 卡片懸停/點擊高亮效果

2. **單字學習面板**
   - 點擊假名展開詳情
   - 顯示該假名開頭的單字（2-3個）
   - 每個單字配有圖片
   - 播放按鈕朗讀假名和單字

3. **語音功能**
   - 使用 Web Speech API (SpeechSynthesis)
   - 選擇日語發音
   - 點擊播放按鈕朗讀
   - 朗讀時顯示脈動動畫

4. **假名類別篩選**
   - 全部 / 清音 / 濁音 / 半濁音 按鈕
   - 即時過濾顯示

### 互動細節
- 點擊卡片：展開學習面板，背景模糊
- 點擊關閉：面板淡出，卡片回位
- 播放語音：按鈕變色 + 脈動動畫
- 滑過卡片：輕微放大 + 陰影加深

### 邊界條件
- 無語音支援時：顯示提示訊息
- 圖片載入失敗：顯示 emoji 替代

## 5. 組件清單

### 假名卡片 (KanaCard)
- 預設：大圓角卡片，白色背景，柔和陰影
- 懸停：scale(1.05)，陰影加深
- 點擊：scale(0.95) 快速回彈
- 顯示：平假名（主要）、片假名、羅馬拼音

### 播放按鈕 (PlayButton)
- 預設：圓形，強調色背景，白色播放圖標
- 播放中：脈動動畫，背景閃爍
- 禁用：灰色，鼠標變為 not-allowed

### 篩選按鈕 (FilterButton)
- 預設：圓角矩形，透明背景
- 選中：填充主色，白色文字
- 懸停：背景加深 10%

### 學習面板 (LearningPanel)
- 背景：白色卡片，居中顯示
- 動畫：scale(0.8) → scale(1) + fadeIn
- 包含：關閉按鈕、大假名、單字列表、圖片

### 單字項目 (WordItem)
- 顯示：圖片 + 日文單字 + 中文翻譯 + 播放按鈕
- 懸停：背景輕微變色

## 6. 技術方案

### 框架
純 HTML + CSS + JavaScript（無需構建工具）

### 關鍵技術
- CSS Grid / Flexbox 響應式佈局
- CSS Variables 管理主題
- Web Speech API 文字轉語音
- CSS Animations 動畫效果
- Media Queries 響應式斷點

### 數據結構
```javascript
{
  kana: "あ",        // 平假名
  katakana: "ア",    // 片假名
  romaji: "a",       // 羅馬拼音
  type: "vowel",     // 類型：vowel/consonant/dakuten/handakuten
  words: [
    { 
      japanese: "りんご",  // 日文
      chinese: "蘋果",     // 中文
      image: "🍎"          // 圖片/emoji
    }
  ]
}
```

### 假名數據
涵蓋所有基本假名：
- 清音 46 個（五十音圖）
- 濁音 25 個（が-ぽ）
- 半濁音 5 個（ぱ-ぽ）

每個假名配 2-3 個常用單字
